LIX of Danish Speeches
========
The speeches.csv file contains the LIX in speeches given by the Danish Prime Minister and the Queen of Denmark. LIX is a readability measure indicating the difficulty of reading a text developed by Swedish scholar Carl-Hugo Bj√∂rnson. It is computed as follows:

LIX = A / B + (C * 100) / A, where
